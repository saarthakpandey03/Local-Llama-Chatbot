# Local Llama3 CLI Chatbot 🤖

A simple terminal-based AI chatbot built using **Python** and **Ollama Llama3** with conversation memory and typing animation.

## Features 🚀

* Local Llama3 model using Ollama
* Conversation memory
* Hinglish responses with emojis
* Typing animation effect
* Clear chat memory command
* Exit commands support
* Lightweight and beginner-friendly

---

## Technologies Used 🛠️

* Python
* Ollama
* Llama3
* Datetime
* Time module

---

## Installation ⚡

### 1. Clone the repository

```bash
git clone https://github.com/your-username/local-llama-chatbot.git
cd local-llama-chatbot
```

### 2. Install dependencies

```bash
pip install ollama
```

### 3. Install Ollama

Download Ollama from:

https://ollama.com

### 4. Pull Llama3 model

```bash
ollama pull llama3
```

### 5. Run the chatbot

```bash
python app.py
```

---

## Commands 💬

| Command | Description           |
| ------- | --------------------- |
| `clear` | Clears chatbot memory |
| `exit`  | Exits chatbot         |
| `bye`   | Exits chatbot         |
| `quit`  | Exits chatbot         |
